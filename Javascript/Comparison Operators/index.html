Comparison Operators

Comparison operators are used in logical statements to determine equality or difference between variables or values.

Given that x = 5, the table below explains the comparison operators:

Operator Description Example
== Equal to x == 8
=== Equal value and equal type x === 5
!= Not equal x != 8
!== Not equal value or not equal type x !== 5
> Greater than x > 8
< Less than x < 8
>= Greater than or equal to x >= 8
<= Less than or equal to x <= 8

For example, the expression x == 8 is true because x is 5.

For example, the expression x === 5 is true because x is 5 and of the same type.

For example, the expression x != 8 is true because x is not 8.

For example, the expression x !== 5 is false because x is 5 and of the same type.

For example, the expression x > 8 is false because x is not greater than 8.

For example, the expression x < 8 is true because x is less than 8.

For example, the expression x >= 8 is false because x is not greater than or equal to 8.

For example, the expression x <= 8 is true because x is less than or equal to 8.
